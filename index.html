Using 3D mouse to control Robot Manipulator

Abstract: We present a web-based software package that simplifies the integration of 3D mice for 6DOF robot teleoperation, enhancing usability with configurable input processing and interactive visualization. The package includes a demonstration interface for seamless integration with ROS/ROS2 systems and practical usage guidance..

I Can Tell What I am Doing: Toward Real-World Natural Language Grounding of Robot Experiences Advances in SLAM Algorithms

Abstract: We introduce RONAR, an LLM-based system that generates natural language narrations from multi-modal robot experiences, improving transparency, behavior analysis, and failure recovery. RONAR outperforms state-of-the-art methods, offering a novel framework and dataset to enhance user interaction and system understanding. Read Paper | Code | Website

AccessTeleopKit: A Toolkit for Creating Accessible Web-Based Interfaces for Tele-Operating an Assistive Robot Reinforcement Learning in Robotics

Abstract: We present AccessTeleopKit, an open-source toolkit enabling customizable and accessible teleoperation interfaces for the Stretch 3 mobile manipulator, designed for users with severe motor limitations. Developed through participatory design and home deployments, it empowers users to create task-specific interfaces, enhancing independence in daily activities.
